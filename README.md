# AvaCalc

## Description

`AvaCalc` is a project that re-writtes implementation of `KCalc`.

## Motivation

The aim of this project is mostly make project consume
less RAM than modern calculators. Modern calculators consumes approximately about 30 MB of RAM. It's justified, when
powered user uses all available tools to achieve all goals, but for average user, even 30 MB of RAM is too much. Yes,
I'm aware that we're in era when 8 GB of RAM is minimal standard. Yet, it  doesn't mean we shoudn't try to optimize it,
like it was done before (calculator in Windows XP uses about 1-2 MB. Yes, I'm aware that it contained less
functionality).

## Used tools

- .NET 10
- Avalonia UI 12
